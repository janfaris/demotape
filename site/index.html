Your entire demo in one JSON file

Define pages, scroll choreography, dwell times, and text overlays. demotape handles the rest.

demo.json 
{
  "baseUrl": "http://localhost:3000",
  "viewport": { "width": 1280, "height": 800 },
  "output": { "format": "both", "name": "demo" },
  "auth": { "provider": "supabase" },
  "segments": [
    {
      "name": "Home",
      "path": "/",
      "waitFor": "h1",
      "dwellMs": 3000
    },
    {
      "name": "Dashboard",
      "path": "/dashboard",
      "waitFor": "main",
      "scroll": { "distance": 400 },
      "dwellMs": 2000
    }
  ]
}

Built for developers who ship

Everything you need to keep your demo videos in sync with your product — automatically.

Skeleton-free

Each segment is recorded separately. Loading frames are measured and trimmed — only fully rendered content makes the cut.

Auth-aware

Record apps behind login. Supports Supabase magic links, raw cookie injection, and localStorage tokens out of the box.

Multi-format

Landscape for landing pages, vertical for Instagram Stories, 16:9 for Product Hunt. One tool, any aspect ratio, any resolution.

CI/CD native

Runs headlessly in GitHub Actions, GitLab CI, or any pipeline. Your demo videos update automatically on every deploy.

Config-driven

Define everything in JSON — pages, scroll distances, dwell times, text overlays. Version control your demo alongside your code.

Text overlays

Burn branded titles and CTAs directly into the video via FFmpeg. Perfect for Instagram Stories and social media clips.
